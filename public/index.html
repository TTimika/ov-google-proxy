🚀 第二步：导入到奥维地图

  1. 打开奥维地图（电脑端或手机端）
  2. 点击顶部菜单 "系统" → "导入对象"
  3. 在弹出的窗口中，选择 "从文本导入"
  4. 粘贴刚才复制的完整 XML 配置内容
  5. 点击 "确定" 完成导入

🌍 第三步：启用 Google 图层

  1. 导入成功后，在左侧图层列表中展开 "谷歌官方"
  2. 勾选 "谷歌卫星" 启用卫星图
  3. 地图会自动切换到 Google 卫星底图
💡 提示：如果加载缓慢，可以右键点击图层 → "删除"，然后重新导入配置。

ℹ️ 技术详情

服务器地址 -
URL 模板 /vt/lyrs=s@699&hl=zh-CN&gl=cn&src=app&x={$x}&y={$y}&z={$z}&s=
支持的 lyrs 参数 • s - 卫星图（默认）
• m - 街道图
• y - 混合图（街道 + 卫星）
最大缩放级别 28
坐标系 Web Mercator (EPSG:3857)
瓦片尺寸 256 × 256 像素

❓ 常见问题

问题 1: 导入后显示"No Data"或空白？
解决: 检查 host 字段是否填写正确（不应包含 https:// 前缀）
问题 2: 地图加载很慢或超时？
解决: Cloudflare Worker 有免费额度限制（10 万请求/月），如超量需更换域名或自建服务
问题 3: 如何切换不同类型的地图？
解决: 修改 XML 中 <url> 里的 lyrs 参数值即可
